# kith-pms

**kith-pms** is a self-hosted Personal Relationship Manager (PRM) — a private, single-user tool for keeping track of the people in your life.

## What it is

- Track contacts, relationships, important dates, and gift history
- Log journal entries linked to people
- Set reminders and follow-ups
- Import from Monica CRM

## Who it's for

Anyone who wants a private, self-hosted alternative to cloud CRMs. No subscriptions, no data sharing — just a single binary and a SQLite file you own.
