# Journal

The journal is a log of interactions and events linked to people.

## Add an entry

1. Go to **Journal** → **New entry**
2. Fill in title, date, and content
3. Link one or more people to the entry
4. Save

You can also add a quick journal entry directly from a person's detail page using the **Quick journal** button in the Journal section.

## Search

Full-text search is available on the Journal list page. Search covers title and content.

## Filter by person

Use the people filter pills on the Journal list to show only entries involving specific people.

## Edit & delete

Open an entry's detail page to edit or delete it.
