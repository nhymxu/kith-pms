# Reminders

Reminders let you schedule follow-ups linked to people and dates.

## Add a reminder

1. Go to **Reminders** → **New reminder**
2. Set a title, due date, and optionally link a person
3. Save

## Complete a reminder

Click the checkmark on a reminder to mark it complete. Completed reminders are hidden from the default view.

## Overdue reminders

Reminders past their due date appear highlighted in the list and in the dashboard Action Queue.
